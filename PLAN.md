# K-ETF Trend Rotation Agent 개발 계획서 (v2 — 최종 확정)

## 1. 프로젝트 정의

### 목표

프로그램 실행 시:

1. 미리 정의된 ETF 종목 데이터 조회
2. 일봉 / 주봉 보조지표 계산
3. 현재 보유 상태(state.json) 확인
4. 매수 / 매도 / 유지 신호 출력

을 자동 수행하는 **로컬 콘솔 기반 ETF 추세 로테이션 시스템** 구축.

### 최종 한 줄 정의

> "일목구름 + 이동평균 + ADX 기반 ETF 추세 로테이션 콘솔 에이전트"

### 대상 사용자

- 본인 단독 사용
- 수동 매매 보조용 (자동 주문 없음)
- 콘솔 출력 전용

## 6. 전략 로직

### 6.1 구름 위/아래 판정 (확정)

- **구름 위**: 종가가 선행스팬1과 선행스팬2 **모두보다 높음** (구름에 걸쳐있지 않음)
- **구름 아래**: 종가가 선행스팬1과 선행스팬2 **모두보다 낮음** (구름에 걸쳐있지 않음)
- **구름 안**: 위 두 조건 모두 아닌 경우 → 시그널 없음

### 6.2 코스피 상승 ETF

#### 1x 매수 조건 (일봉, AND)

전일 종가 기준, **3가지 조건 모두 충족** 시:

1. 종가가 일목구름 완전 상단 (구름 위)
2. 종가 > 60MA
3. 선행스팬1 > 선행스팬2 (골든크로스)

→ **KODEX 200 매수 (BUY)**

#### 2x 전환 조건 (주봉, AND)

전일 종가 기준:

1. 주봉 종가가 주봉 일목구름 완전 상단
2. 주봉 종가 > 주봉 60MA

→ **KODEX 200 매도 + KODEX 레버리지 매수**

#### 매도 조건 (일봉, OR)

전일 종가 기준, **어느 하나라도 충족** 시:

1. 종가가 일목구름 완전 하단 (구름 아래)
2. 종가 < 60MA

→ **전량 매도 (SELL)** (1x든 2x든 무관)

> ⚠️ 2x → 1x 전환은 없음. 매도는 항상 일봉 기준 전량 매도.

### 6.3 코스피 인버스 ETF

#### 1x 매수 조건 (일봉, AND)

**KODEX 인버스 자체 지표** 기준:

1. 종가가 일목구름 완전 상단
2. 종가 > 60MA
3. 선행스팬1 > 선행스팬2

→ **KODEX 인버스 매수 (BUY)**

#### 2x 전환 조건 (주봉, AND)

KODEX 인버스 자체 주봉 지표 기준:

1. 주봉 종가가 주봉 일목구름 완전 상단
2. 주봉 종가 > 주봉 60MA

→ **KODEX 인버스 매도 + KODEX 200선물인버스2X 매수**

#### 매도 조건 (일봉, OR)

KODEX 인버스 자체 지표 기준:

1. 종가가 일목구름 완전 하단
2. 종가 < 60MA

→ **전량 매도**

### 6.4 나스닥 상승 ETF

#### 1x 매수 조건 (일봉, AND)

KODEX 미국나스닥100 지표 기준:

1. 종가가 일목구름 완전 상단
2. 종가 > 60MA
3. 선행스팬1 > 선행스팬2

→ **KODEX 미국나스닥100 매수 (BUY)**

#### 2x 전환 조건 (주봉, AND)

1. 주봉 종가가 주봉 일목구름 완전 상단
2. 주봉 종가 > 주봉 60MA

→ **KODEX 미국나스닥100 매도 + KODEX 미국나스닥100레버리지(합성 H) 매수**

#### 매도 조건 (일봉, OR)

1. 종가가 일목구름 완전 하단
2. 종가 < 60MA

→ **전량 매도**

### 6.5 나스닥 인버스 ETF (역방향 해석)

> 나스닥 인버스 1x ETF가 없으므로, **KODEX 미국나스닥100의 지표를 역방향으로 해석**

#### 하락 추세 판단 (일봉, AND)

KODEX 미국나스닥100 기준:

1. 종가가 일목구름 완전 하단 (구름 아래)
2. 종가 < 60MA
3. 선행스팬1 < 선행스팬2 (데드크로스)

→ **나스닥 하락 추세 판단**

#### 인버스 진입 조건 (주봉, AND)

KODEX 미국나스닥100 기준:

1. 주봉 종가가 주봉 일목구름 완전 하단
2. 주봉 종가 < 주봉 60MA

→ **KODEX 미국나스닥100선물인버스(H) 매수**

#### 인버스 매도 조건 (일봉, OR)

KODEX 미국나스닥100 기준:

1. 종가가 일목구름 완전 상단 (구름 위)
2. 종가 > 60MA

→ **KODEX 미국나스닥100선물인버스(H) 전량 매도**

### 6.6 충돌 해결

같은 시장 내 상승 ETF와 인버스 ETF 시그널 충돌 시:

- **ADX >= 25** → 신규 추세 우선 (새 시그널 채택)
- **ADX < 25** → 기존 포지션 유지 (STAY)

> 코스피와 나스닥 간 충돌은 없음. 각 시장 독립 판단, 사용자가 자금 배분.

### 6.7 시그널 종류

| 시그널 | 의미 |
|--------|------|
| BUY | 매수 |
| SELL | 매도 |
| STAY | 충돌 시 기존 포지션 유지 |
| UPGRADE | 1x → 2x 전환 (1x 매도 + 2x 매수) |
| None | 시그널 없음 (전체) |

## 7. 포지션 상태 관리

### state.json

```json
{
  "KODEX200": false,
  "KODEX_LEVERAGE": false,
  "KODEX_INVERSE": false,
  "KODEX_INVERSE_2X": false,
  "KODEX_NASDAQ100": false,
  "KODEX_NASDAQ100_LEVERAGE": false,
  "KODEX_NASDAQ100_INVERSE": false
}
```

- 초기값: 전부 `false`
- 사용자 직접 수정
- 같은 시장 내 상승+인버스 동시 `true`는 사용자 실수 → 경고 출력

### 출력 규칙

- **시그널이 있는 종목만 표시**
- 충돌 시 STAY도 시그널에 포함하여 표시
- 모든 종목에 시그널 없으면 `None` 출력
